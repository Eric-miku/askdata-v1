# AskData 产品使用说明

## 产品定位

AskData 面向不会编写 SQL 的业务人员。用户选择已授权的数据源，以自然语言提出统计问题，系统完成问题理解、Schema 检索、业务口径匹配、只读 SQL 生成、安全校验、执行、表格/图表展示和结构化分析。

当前可运行版本以 BIRD 和管理员注册的受控 SQLite 数据源为主。公司 MySQL/PostgreSQL、SSO、动态角色策略和正式业务指标口径属于部署侧接入范围。

## 主要角色

| 角色 | 能力 |
| --- | --- |
| 业务用户 | 选择授权数据源、自然语言提问、多轮追问、查看 SQL/图表/分析、执行授权范围内的导出 |
| 数据管理员 | 维护术语和指标、注册与同步数据源、查看 Schema Catalog、配置对象权限、检查执行计划 |
| 运维人员 | 配置模型和状态目录、部署容器、检查健康/就绪/指标、使用 request ID 与 trace ID 排障 |

生产环境中的用户身份必须由可信网关或 SSO 注入 `X-User-ID`；浏览器自行提供的身份只能用于本地开发。

## 问数流程

1. 在左侧数据库选择器中选择当前用户有权访问的数据源。
2. 输入问题，例如“上个月按部门统计销售额前 5 名，只看华东”。
3. 查询期间可在 Trace 中查看结构化理解、Schema 检索、SQL 执行和修复状态。
4. 成功后检查文字回答、生成 SQL、结果表、推荐图表和可追溯分析结论。
5. 使用图表工具栏切换柱状图、条形图、折线图或饼图；不适合构图时保留表格。
6. 使用 SQL 面板的“执行计划”查看只读 `EXPLAIN QUERY PLAN` 和人工索引候选。
7. 使用 CSV、Excel 或图表 PNG 导出。服务端导出会重新执行当前权限检查。

SQL 面板展示的索引 SQL 仅供数据库管理员评估，AskData 不会自动修改数据库索引。

## 多轮追问

同一会话保存指标、维度、筛选、时间范围、排序、TopN、SQL、结果摘要和数据源。支持：

- “改成上个月”：替换时间范围。
- “只看华南”：替换排他地区筛选。
- “按部门查看”：增加或替换维度。
- “换成客户数”：替换指标。
- “不限制地区”“取消部门分组”：显式清除条件。

切换数据源会清除不兼容的历史 SQL；不同用户和会话不能互相读取上下文。

## 术语与指标管理

打开左侧术语入口后，可以维护标准名、定义、分类、范围、别名、字段映射、公式、聚合方式、单位、时间字段、示例问法和版本说明。

- 草稿可以编辑和批量导入。
- 发布前会校验映射的数据源、表和字段。
- 已发布别名和口语表达会参与查询匹配。
- 相同术语存在冲突口径时，系统要求用户澄清。
- 支持历史版本查看和回滚。
- JSON 批量导入最多 1000 条，逐条返回错误；支持 JSON/CSV 导出。

## 数据源与权限管理

数据源管理页面支持受控 SQLite 的新增、连接测试、启停、删除和 Schema 同步。同步结果包含表、列、DDL、主键、外键、索引、SHA-256 指纹和相对上次同步的变更摘要。

对象权限按用户配置数据源、表、字段、查询和导出白名单。表级策略还可配置受限的行过滤表达式，例如 `region = '华东'`；系统以 SQL AST 将策略注入每个受控物理表，查询、SQL 回放、执行计划和导出统一执行，展示给用户的 SQL 不包含内部策略。配置第一条策略后进入白名单模式。

行过滤只允许列、字面量、比较、布尔组合、`IN`、`BETWEEN`、`LIKE` 和 `IS NULL` 等受限表达式，不允许函数、子查询、多语句或引用其他表。生产环境仍应由可信 SSO 身份映射到这些策略。

## 状态与错误

| 状态 | 用户动作 |
| --- | --- |
| 未选择数据源 | 先在数据库选择器中选择数据源 |
| 语义歧义 | 按澄清问题确认指标口径、日期字段或术语 |
| SQL 修复中 | 查看 Trace，等待有限次数的自动修复 |
| 权限拒绝 | 联系管理员核对数据源、表、字段或导出策略 |
| 查询超时/结果过大 | 缩小时间范围、增加筛选或降低明细粒度 |
| 外部模型不可用 | 使用 request ID 联系运维检查模型网关 |

普通用户响应不包含数据库堆栈、凭据或内部路径。排障时应提供响应头中的 `X-Request-ID`。

## 当前边界

- SQLite 没有原生字段注释元数据，Catalog 保留 DDL，但不推断不存在的注释。
- OLS 预测用于跑通分析流程，不替代经过业务验证的生产时间序列模型。
- PNG 内容和响应式布局仍需在部署浏览器中执行像素级验收。
- 正式准确率必须使用公司固定题集、真实模型和已签字的指标口径测量。
